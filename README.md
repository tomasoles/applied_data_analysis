# Applied Data Analysis
Slides and exercises for the course Applied Data Analysis.

The course is structured in weekly modules that follow the syllabus_ADA_en.pdf. Please note that this is subject to change.

Each week contains the necessary data files and two .Rmd files. For example, 1_week_p.Rmd contains the editable and reproducible presentation for the first week's lecture, which generates a reproducible 1_week_p.html presentation and analysis from the lecture.

The *_week_p and *_week_e files correspond to the lecture and exercise components, respectively.

I am very thankful to @datascience-box and everyone in the #rstats education community for providing numerous exercises and teaching materials that are freely available and open source, which form the foundation of this course.
