# Applied Data Analysis
Slides and exercises for the course Applied Data Analysis.

The course is structured in weeks that follows the ~syllabus_ADA_en.pdf. This is subject to change. 

Each week contains necessary data file and two .Rmd files. For example the 1_week_p.Rmd contains the editable and reproducible presentation for lectures 1st week, which creates a reproducible 1_week_p.html presentation and analysis from the lecture. 

The *_week_p - *_week_e for the  and  is appendix for lectures and exercise part, respectively. 
I am very thankful to @datascience-box and everyone in the #rstats education community for providing numerous exercises and teaching materials that are freely available and open source, which form the foundation of this course.
